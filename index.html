What I study

Research Focus

I work at the intersection of neural dynamics, computational modeling, and social decision-making. The threads below anchor my current questions.

Attractor Dynamics

How stable memory states form, persist, and drift under noise, uncertainty, and task demands.

Reinforcement Learning

Modeling how valence and uncertainty shape learning, with emphasis on adaptive strategies and bias.

Physiological Synchrony

Exploring interpersonal heart rate synchrony and its impact on group decision-making and performance.

Open source

Projects & Packages

Open-source tools and reproducible workflows for complex time-series analysis.

SMdRQA

A Python package for Sliding Window Multidimensional Recurrence Quantification Analysis. It helps capture temporal patterns and dependencies in multidimensional data.

Documentation PyPI
  • Sliding window analysis of multidimensional time series
  • Multiple recurrence quantification measures
  • Efficient computation using NumPy
  • Comprehensive documentation with examples
  • Compatible with the SciPy ecosystem

Example Usage

from SMdRQA.RP_maker import RP_computer
from SMdRQA.Extract_from_RP import windowed_RP
from SMdRQA.Extract_from_RP import First_middle_last_sliding_windows_all_vars

RP_computer(input_path, RP_dir)

Dict_RPs = windowed_RP(window_size, "RP")
First_middle_last_sliding_windows_all_vars(Dict_RPs, "example_data.csv")

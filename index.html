Introduction

My Academic Journey

Raised in the heart of a village in Kerala, my academic journey emerged from contemplations on the meaning of self, life, and the enigma of reality. What particularly fascinated me was the intricate interplay between financial decisions, especially in a middle-class setting, and the pervasive spread of superstitious beliefs.

The juxtaposition of significant life choices made with little deliberation alongside the propagation of unfounded beliefs challenged my understanding of objective truth. It compelled me to reevaluate reality through the lens of subjective beliefs, resembling the adaptive nature of a Bayesian model. This dual fascination forms the core of my research interests in neuroscience, decision-making, and cognitive modelling.

I first fell for physics during my early days at IISER Pune. In my second year, I found myself wondering if there might be something even deeper than the physical reality we probe with equations and experiments—what truly shapes the world we perceive? A conversation with a humanities professor led me to Carlos Castaneda’s Teachings of Don Juan.

Codes & Packages

SMdRQA

SMdRQA is a Python package designed for conducting Sliding Window Multidimensional Recurrent Quantification Analysis (SMdRQA). This specialized analysis technique allows researchers to explore temporal patterns and dependencies in multidimensional data sequences using a sliding window approach.

Documentation PyPI 
# Example usage of SMdRQA
from SMdRQA.RP_maker import RP_computer

from SMdRQA.Extract_from_RP import Mode
from SMdRQA.Extract_from_RP import Check_Int_Array
from SMdRQA.Extract_from_RP import Sliding_window
from SMdRQA.Extract_from_RP import Whole_window
from SMdRQA.Extract_from_RP import windowed_RP
from SMdRQA.Extract_from_RP import First_middle_last_avg
from SMdRQA.Extract_from_RP import First_middle_last_sliding_windows
from SMdRQA.Extract_from_RP import First_middle_last_sliding_windows_all_vars

# Create RPs
RP_computer(input_path, RP_dir)    # generating RPs and saving to the specified folder


# Compute RQA metrics and save
Dict_RPs=windowed_RP(window_size, 'RP')                                                      # Specifying window size and folder in which RPs are saved
First_middle_last_sliding_windows_all_vars(Dict_RPs,'example_data.csv')
